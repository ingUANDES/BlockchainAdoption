A partir del modelo de Jalan de 3 agentes en una red, agregamos un cuarto agente (una fintech) y luego una segunda red (blockchain). Se estudian los casos en los que la nueva fintech está totalmente restringida y luego se agrega la restricción original entre el banco nacional y la firma local. 


Los escenarios se realizaron con parámetros neutrales en la segunda red($\tau = 0$, $\lambda_B = 0$).

\section{Configuración Original}

\subsection{Agentes}
\begin{itemize}
\item Agente 0: Banco Nacional
\item Agente 1: Banco Local  
\item Agente 2: Firma Local
\item Agente 3: Fintech (solo en modelos de 4 agentes)
\end{itemize}

\subsection{Parámetros del Modelo}

\begin{equation}
