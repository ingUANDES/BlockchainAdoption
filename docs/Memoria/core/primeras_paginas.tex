  \begin{center} \Large \textbf{Abstract} \end{center}

Regulatory restrictions in financial networks create welfare losses by prohibiting efficient contracting relationships. This thesis investigates whether alternative network infrastructures can restore welfare by enabling agents to circumvent such restrictions while traditional constraints remain in force. Extending the framework of \citeA{jalan_incentive-aware_2024} to dual-network stability, we model agents who optimize portfolios across traditional and alternative infrastructures. Using a three-agent network where a regulatory restriction prohibits one link, we demonstrate that alternative network adoption achieves complete welfare recovery, restoring aggregate utility. In contrast, promoting competition within the restricted network recovers only a part of lost welfare, leaving a persistent efficiency gap. The findings suggest that regulatory arbitrage, not inherent efficiency, drives adoption in restricted markets. Where restrictions serve legitimate objectives, regulators face a trilemma among maintaining constraints, preventing circumvention, and accepting welfare losses. The dual-network framework provides analytical tools for evaluating parallel infrastructure competition.
